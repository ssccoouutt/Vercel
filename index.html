üìö API Documentation

Use this site as an API to get video information and download links programmatically.

üì° Base URL

https://vercel-steel-pi.vercel.app

üîç Get Video Information

Returns metadata about a YouTube video without downloading.

Endpoint: GET /?api=info&url=VIDEO_URL Example: https://vercel-steel-pi.vercel.app/?api=info&url=https://youtu.be/dQw4w9WgXcQ

üì• Get Download Link

Returns a direct download link for the video/audio.

Endpoint: GET /?api=download&url=VIDEO_URL&format=video&quality=720p Parameters:
  • url (required) - YouTube video URL
  • format (optional) - 'video' or 'audio' (default: video)
  • quality (optional) - 'highest', '1080p', '720p', '480p', '360p' (default: 720p)
Example: https://vercel-steel-pi.vercel.app/?api=download&url=https://youtu.be/dQw4w9WgXcQ&format=video&quality=720p

üíª Usage Examples

Python
import requests # Get video info response = requests.get('https://vercel-steel-pi.vercel.app/', params={'api': 'info', 'url': 'https://youtu.be/dQw4w9WgXcQ'}) data = response.json() print(f"Title: {data['title']}") # Get download link response = requests.get('https://vercel-steel-pi.vercel.app/', params={'api': 'download', 'url': 'https://youtu.be/dQw4w9WgXcQ', 'format': 'video', 'quality': '720p'}) data = response.json() print(f"Download URL: {data['download_url']}")
